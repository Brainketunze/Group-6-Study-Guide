What is JavaScript?

JavaScript was initially created to “make web pages alive”.
The programs in this language are called scripts. They can be written right in a web page’s HTML and run automatically as the page loads.
Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run.
In this aspect, JavaScript is very different from another language called Java.

Logical Operators

OR || the or operator does not evaluate to true or false!
it returns the left value or the right value
- if the left is truthy, it returns the left value
- if the left is falsy, it returns the right value

And && the and operator does not evaluate to true or false!
it returns the left value or the right value
- if the left is falsy, it returns the left value
- if the left is truthy, it returns the right value

Not ! the not operator does evaluate to true or false
- if the value is truthy, it returns false
- if the value is falsy, it returns true

Block Scope

Block Scope: const
writing curly braces creates a new block scope
block scopes only effect variables
everything else you've learned so far still applies
here are few rules to keep in mind
1. variables declared outside a block:
a. are available inside the block if there is not a block variable with the same name
b. are not available inside the block if there is a new variable of the same name
2. variables declared inside a block:
a. are available inside the block
b. are not available outside the block
c.'replace' variables declared outside
`let` and Block Scope
 all of the rules you learned in the last example apply to `let` variables
plus these new rules:
3. `let variables declared outside of a block ...
a. can be modified inside the block if they are not replaced
b. cannot be modified inside the block if they are replaced
